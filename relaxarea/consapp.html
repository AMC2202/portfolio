Gioca con la versione scritta in python

Guida all'uso

  1. Apri il terminale e verifica che python sia installato sul tuo pc, digitando il comando python --version;
    • Se non ti viene restituito il numero di versione di python, significa che non è stato ancora installato;
    • Apri il browser e recati sulla pagina ufficiale di python per scaricare l'ultima versione stabile disponibile;
    • Fai doppio click sul pacchetto appena scaricato per aprire il wizard e procedere all'installazione;
    • Apri il terminale e verifica che python sia stato correttamente installato sul tuo pc, digitando il comando python --version;
  2. Se ti viene restituito il numero di versione di python, significa che è già stato installato sul tuo pc;
  3. Clicca sull'icona a sinistra per scaricare la cartella compressa. Decomprimi, e copia il percorso di quest'ultima cartella
  4. Digita da terminale il comando cd e incolla il percorso per spostarti nella cartella che contiene il file .py;
  5. Adesso che ti trovi nel percorso dedicato, lancia il programma da riga di comando, digitando python e il nome del file. Assicurati di specificare l'estensione .py dello stesso.
